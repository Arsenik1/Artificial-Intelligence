Program description

Environment:

There are pendulums (one or more) on the mobile cart. The cart can move left or right. The purpose of steering is to keep the pendulums vertical (or as close to vertical as possible) and the trolley at the zero point for a given number of simulation and control steps.

Manual control is possible.


Program functions:

Key operation

KEY

FUNCTION

Escape

Quit from a program

End

positioning the trolley in the center of the window (continuous mode with the Space key)

Space

enabling / disabling tracking mode (in tracking mode, the view is set so that the layout is in the center of the window)

Backspace

restoration of zero position

TAB or O

scene rotation on / off

Right mouse button

applying force on the right side of the trolley (using the keys 1,2, ..., 9 you can adjust the amount of force, e.g. key ‘1’ allows you to use 10% of the maximum force)

Left mouse button

force is applied to the left side of the trolley

PageUp, PageDown

zoom in and out the view

arrows

moving the view up-down, left-right

K

turning on / off the control mode (force value selection) with the mouse cursor. After switching on the mode, 3 vertical lines appear in the window: the center line corresponds to the zero force, the outermost lines correspond to the maximum values of the force applied to the left and right respectively. When you apply force, you need to press the left mouse button. When the mode is not on, the force direction depends on the use of the left or right mouse button, and the force value can be selected using the 1 ... 9 keys.

H

enable / disable history recovery mode (there must be a history.txt file in the directory)

Z

Previous initial state from history

X

Next initial state from history
